Svelte Kit is the successor to Sapper

Prior to this year, if you wanted to build a highly perfomant, full featured web app featuring server side rendering and file based routing with Svelte as the underlying view library, you would need to use Sapper. However, at the beginning of the year, Rich Harris announced that Sapper would never make it to version 1.0 and announced a new initiative called SvelteKit.

Now, SvelteKit is approaching a full release and, while still in beta, can provide us with the next level of developer experience for Svelte developers.

Already, there are a lot of great resources for learning SvelteKit and there are even official guides for migrating Sapper apps to SvelteKit. So, while the project is still in beta, it is a great time to get familiarized with SvelteKit so you can be ready to jump in with both feet once version 1.0 is officially launched.
